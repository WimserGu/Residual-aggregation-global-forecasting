# Reproducibility checklist

## Data

- Required input: `data/processed/monthly_market_series.csv`
- Required columns: `date`, `brand`, `sales`, `market_size`
- Raw data are not redistributed by default because redistribution rights must be checked separately.
- Synthetic smoke-test data can be generated with:

```bash
python scripts/prepare_processed_data.py --synthetic-sample --output data/processed/monthly_market_series.csv
```

## Experiment settings

- Forecast horizon: one month ahead.
- Evaluation: rolling-origin evaluation.
- Initial training window: first 48 months.
- Target evaluation: selected observed brands only.
- Top-K values: `5`, `8`, `10`, `12`, `14`.
- Base random seed: `20260623`.
- Multi-seed placebo replications: `20`.
- Bootstrap replications: `10000`.

## Expected runtime

Runtime depends on hardware and LightGBM installation. As a rough guide:

- Coverage ladder: minutes.
- Representation ablation: minutes.
- Multi-seed placebo validation: substantially longer because it repeats rolling-origin LightGBM fitting across K values and seeds.
- Figure and table generation: seconds.

Use `python scripts/run_all_experiments.py --skip-placebos` for a quick workflow check.

## Output separation

- Archived manuscript outputs are stored in `results/manuscript/` and `figures/manuscript/`.
- Rerun outputs are written to `results/reproduced/` and `figures/reproduced/`.
- The reproduction scripts should not overwrite archived manuscript outputs.

## Scope

The public-release workflow reproduces the final residual-aggregation manuscript.
Earlier exploratory market-structure experiments are intentionally excluded from
the clean reproduction path.
